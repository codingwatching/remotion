This is a list of features that are available in the [Editor Starter](https://remotion.pro/editor-starter).

## Timeline

- Multiple tracks
  - Plus button to add a new track
- Time ticks
- Zoom slider
- Draggable playhead
  - Dragging the playhead to the edge will scroll the timeline
- Filmstrip thumbnails (`FEATURE_FILMSTRIP`)
- Drop asset to add new layer
  - Detection of item type

## Layer controls

- Layer alignment controls
  - Left, vertical center, right
  - Top, horizontal center, bottom
- Position controls
- Size controls
- Border radius control (`FEATURE_BORDER_RADIUS_CONTROL`)
- Opacity control (`FEATURE_OPACITY_CONTROL`)
- Text alignment controls
- Font controls
  - Font family
    - Dropdown with font previews
  - Font size
- Fill control

## Playback

- Play / Pause
  - Keyboard shortcut `Space`
  - Play / Pause button
- Current time indicator
- Fullscreen button (`FEATURE_FULLSCREEN_BUTTON`)
- Mute button
- Loop button

## Canvas

- Click to select item
  - Layer Size indicator
  - Drag selected item to reposition it
  - Drag corners of selected item to resize
- Drop asset to add new layer
  - Detection of item type
  - Upload to S3
    - Progress indicator
      - Circular
      - File size indicator
- Zoom
  - Fit video to viewport
  - Pinch to zoom
  - Ctrl + mouse wheel to zoom
  - Plus and minus buttons
  - Keyboard shortcuts `+` and `-` to zoom in and out
  - Keyboard shortcut `0` to reset zoom

## Item types

- Text
- Solid (`FEATURE_SOLID_ITEM`)
- Video
- Audio
- Image
- GIF
